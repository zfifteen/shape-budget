# Experiments

This folder contains the experimental evidence for the Budget Governor Principle as a control-parameter program.

Core theory and synthesis notes stay at this root.
Each experiment lives in its own folder with:

- `README.md` for the write-up
- `run.py` for the executable artifact
- `outputs/` for figures and data products
